# Claude Code インストールガイド

Claude Code（CLI版）のインストールからセットアップまでの完全ガイドです。

## 概要

このリポジトリには、Claude Code のインストールで困らないための詳しいドキュメントが含まれています。

## ドキュメント

- [Claude_Code_インストールガイド.md](./Claude_Code_インストールガイド.md) - 詳細なインストール手順とトラブルシューティング

## 主な内容

- 前提条件の確認
- インストール方法（グローバル／npx）
- 初回セットアップ手順
- よくあるエラーと対処法
- トラブルシューティングのヒント

## 対象者

- Claude Code を初めてインストールする方
- インストール時にエラーが発生して困っている方
- コマンドの実行方法が分からない方

## クイックスタート

```bash
# グローバルインストール
npm install -g @anthropic-ai/claude-code

# 起動
claude-code
```

詳しくは[インストールガイド](./Claude_Code_インストールガイド.md)をご覧ください。
